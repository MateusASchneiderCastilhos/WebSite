Este é um site pessoal experimental que tem por objetivo deixar registrado os dados que estão sendo gerados durante o processo de obtenção do título de Mestre em Engenharia Elétrica!

Sobre

Resumidamente, o estudo realizado consiste em desenvolver um algoritmo de decomposição de sinais de eletromiografia (EMG, do inglês Electromyography) de alta densidade, baseado no trabalho de (NEGRO et al., 2016), para extrair estimativas das atividades de algumas unidades motoras (MU, do inglês Motor Unit).

O algoritmo é constituído de três etapas: Pré-processamento, Separação e Refinamento. Na etapa de pré-processamento os seguintes procedimentos são utilizados:

  • Retirada das observações dos chamados bad channels, canais que não são adequados para decomposição. A identificação dos bad channels é realizada pelo experimentador, mas a retirada é feita pelo algoritmo.
  • Estender cada observação por um fator constante para aumentar a razão entre o número de observações e o número de fontes.
  • Remover a média das observações estendidas.
  • Branquear as observações para fazer com elas se tornem descorrelacionadas e possuam variância unitária.

As etapas de separação e refinamento incorporam duas técnicas consolidadas na área da decomposição. A primeira etapa inclui a técnica de Análise de Componentes Independentes (ICA, do inglês Independent Component Analysis), que foi implementada com base no algoritmo fastICA proposto por (HYVÄRINEN; OJA, 1997). Entretanto, a função custo (ou contraste) utilizada não se baseia na hipótese das MU’s serem independentes como no algoritmo original, mas analisa a esparsidade, que é uma característica intrínseca das fontes do problema em questão, como proposto por (NEGRO et al., 2016).

A etapa de refinamento é baseada na técnica proposta por (HOLOBAR; ZAZULA, 2007) de Compensar os Kernels intrínsecos do modelo. Essa técnica nomeada de CKC (do inglês, Convolution Kernel Compensation) foi implementada porque a ICA pode fornecer estimativas incertas das fontes, assim, ela visa refinar as fontes estimadas após a execução do algoritmo fastICA. Os dados utilizados pelo CKC primeiramente passam por um algoritmo de detecção de picos e pelo algoritmo de classificação K-means++ proposto por (ARTHUR; VASSILVITSKII, 2007).
